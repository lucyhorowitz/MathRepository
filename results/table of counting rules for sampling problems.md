Choosing $k$ objects from a set of $n$ objects:

| **order counts?** | **repetition allowed?** | **number**           |
|-------------------|-------------------------|----------------------|
| yes               | yes                     | $n^k$                |
| yes               | no                      | $P(n,k) = n!/(n-k)!$ |
| no                | yes                     | ${n+k-1\choose k}$   |
| no                | no                      | ${n\choose k}$       |
